# Website creation for a real estate agency

This is a personal project. My goal is to create a full website for a real estate agency, with a graphic identity and basical features including : a slider on every property listing, filters to get the available houses and flats.

Stack (basic) : HTML, CSS, Javascript.

This is a new challenge for me to get better with JS and maybe including something else like a blog part, but the main goal is to build a nice and branded website with at least one slider and a filter part for the properties.

## The design

I LOVE designing stuffs. Soo i decided to be 100% free on this project. I started by some searches on pinterest and Behance to look at similar projects (real estate websites) and noticed some trends (including a lot of similar palettes for the colors like blue, white, black, grey) and i wanted to do something slightly different with my own preferences. I decided to go for a nice pastel green and some coral, off-white and a deep red colored black. I can't wait to see the result. I am also still thinking to the futur logo, etc. For now i choose Bree serif and Lato as typographies but I am still reconsidering this choice. Looking forward...
